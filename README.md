# 🎭 תיאטרון סימבה — Simba Theater

אתר ראווה אימרסיבי בהשראת מדע בדיוני לתיאטרון סימבה. חוויית גלילה אחת עם ממשקים הולוגרפיים מרחפים, סימולציית חלקיקים, סביבת תלת-ממד וגלריית הצגות אינטראקטיבית.

**🌐 לצפייה חיה:** https://haimgo.github.io/simba-theater/

---

## ✨ מה יש כאן
- **רקע WebGL חי** — ערפילית מבוססת shader, שדה חלקיקים זוהר ואיקוסהדרון הולוגרפי (Three.js)
- **גלריית הצגות** — כרטיסי glassmorphism מרחפים עם הטיית 3D, ומודאל פרטים קולנועי לכל הצגה
- **סינון וחיפוש** — לפי ז'אנר / חודש + חיפוש חופשי (עם התעלמות מניקוד)
- **אווירה קולנועית** — רקע כהה, גרדיאנטי ניאון, סמן מותאם, grain/scanline ומעברים חלקים
- עברית מלאה (RTL), רספונסיבי, תמיכה ב-`prefers-reduced-motion` ו-fallback ללא WebGL

## 🛠️ טכנולוגיה
קובץ יחיד `index.html` — **ללא שלב build**. ‏Three.js נטען מ-CDN. נתוני ההצגות (6 לדוגמה) נמצאים במערך `SHOWS` בראש ה-`<script>` ומיועדים להחלפה בנתונים אמיתיים (לכל הצגה: `title, tagline, genre, synopsis, iso, date, time, duration, hall, price, c1, c2`).

## 🚀 הרצה מקומית
פותחים את `index.html` בדפדפן, או מגישים דרך שרת סטטי:

```bash
npx serve .
```
